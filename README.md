# Data Structures & Algorithms in TypeScript

A comprehensive collection of **custom implementations of Data Structures and Algorithms** built entirely from scratch using **TypeScript**.

The primary goal of this repository is **learning**, not simply using built-in libraries. Every data structure and algorithm is implemented manually to understand its internal design, performance characteristics, and real-world applications.

---

## 🎯 Project Goals

This repository focuses on:

- Understanding how data structures work internally
- Building every data structure from scratch
- Learning algorithm design and problem solving
- Analyzing time and space complexity
- Writing clean, readable, and maintainable TypeScript
- Documenting concepts with notes and examples
- Creating a personal reference for future projects and interview preparation

# Learning Philosophy

This repository follows a simple principle:

> **Don't memorize. Understand.**

Instead of relying on JavaScript's built-in objects such as `Array`, `Map`, or `Set`, we implement everything ourselves to understand:

- Why it exists
- How it works
- When to use it
- When not to use it
- Its advantages and limitations

# Learning Workflow

For every topic, follow this order:

1. Read the notes.
2. Understand the theory.
3. Study the implementation.
4. Run the examples.
5. Modify the implementation.
6. Analyze the complexity.
7. Solve practice problems.

This approach emphasizes understanding rather than memorization.
